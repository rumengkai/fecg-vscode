<h1 align="center">fecg 🚀</h1>

## 代码生成器：Front end code generator

### 使用文档： [FECG](https://temp-static-domain.jd.com/fecg-docs)

### 使用前，先在本地全局安装 fecg
```sh
npm install fecg -g
```
### 在项目工程文件夹右击，点击 `FE Code Generator`,选择要生成的页面类型
